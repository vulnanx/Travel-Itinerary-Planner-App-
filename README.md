## Description
TripPals is a travel planning application, that allows users to create and plan trips with dedicated itineraries. It also allows users to make new friends and invite their newly made friends through their trips. 

## Installation Guide:  <br/>

#### Clone the repository
```bash
git clone https://github.com/CMSC23-UV-1L-AY2024-2025-2ndSem/project-trippals.git
```

#### Move to the directory where you cloned the repository and run the app
```bash
flutter run
```

## How to use the app: <br/>

#### 1. Sign up via email or via google account.
#### 2. Explore the application, make new friends create a travel plan.
#### 3. Share your travel plan with your new friends! 
#### 4. Explore the world through TripPals
